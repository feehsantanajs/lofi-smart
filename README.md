## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/felipesantanajs/lofi-smart.git
```

2. Then go to the project's folder:

```sh
cd lofi-smart

```

3. Install all dependencies:

```sh
npm i
```

4. Run locally:

```sh
npm run dev
```
